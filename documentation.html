Documentation

This page provides a structured overview of the SOMA data model, based on CIDOC CRM and its extensions (CRMarchaeo, CRMsci, ICON, Schema). It defines the ontology classes used for representing archaeological entities and their semantic relationships, distinguishing between core domain elements and supporting generic constructs.
Interactive diagrams illustrate class-property relationships. All nodes are clickable: internal links point to other sections of this documentation, while external references connect to authoritative ontology definitions.

Properties
  • crm:P7_took_place_at
    • Description: Specifies the place where an event, activity, or process occurred. In the SOMA context, this property links activities in the excavation area.
    • Value: Place
  • crm:P8_took_place_on_or_within
    • Description: Indicates the physical object on or within which an event or activity occurred. In the SOMA context, this property can link a cemetery, tomb, grave, burial or object.
    • Value: Cemetery | Tomb | Grave | Burial | Object
  • crm:P4_has_time-span
    • Description: Connects an event or activity to the temporal interval during which it occurred.
    • Value: Time-Span
  • crm:P14_carried_out_by
    • Description: Identifies the actor (person or group) who performed or executed an event or activity. In the SOMA context, this property can link excavations or archaeological activities to the responsible individuals or institutions, such as archaeologists, excavation teams, or cultural organizations.
    • Value:
      • Type: crm:E21_Person
  • crmarchaeo:AP4_produce_surface
    • Description: Indicates that an archaeological modification or activity resulted in the creation of a surface.
    • Value: Stratigraphic Unit
  • crm:P134i_was_continued_by
    • Description: Denotes that an event or activity was followed and extended by another event. In the SOMA context, this property can be used to express how a sequence of archaeological activities relates to each other - for example, the cemetery excavation was continued by the tomb excavation.
    • Value: Archaeological Excavation
  • crm:P70i_is_documented_in
    • Description: Links an entity to a document that records or provides evidence about it. In the SOMA context, this property connects cemeteries, tombs, graves, human remains, or artefacts to excavation permits, archival records or publications where they are described.
    • Value: Document
